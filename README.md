# ddjl
钉钉捡漏机器人

#环境初始化

* 安装python3+ （安装虚拟环境并激活）
* python -m venv venv3 安装虚拟环境
* 激活虚拟环境 
  * windows 
    ```
    ./venv3/Scripts/activate 
  * mac 
    ```
    source venv3/bin/activate
  *  pycharm 
     ```
     可在setting/项目名/python interpreter配置）
* 下载pip并安装
* pip install requests

#运行

python main.py
